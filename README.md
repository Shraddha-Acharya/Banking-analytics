\# 🏦 Indian Banking \& Financial Health Analytics



\## 🛠️ Tech Stack

Python | SQL | Excel | Tableau | Power BI | Scikit-Learn | Git



\## 📌 Overview

End-to-end banking analytics project analyzing Indian banking

and financial data to uncover customer behavior, risk patterns

and financial health insights.



\## 🎯 Business Questions Answered

\- Which customers are financially healthy vs at risk?

\- What spending patterns predict loan defaults?

\- How does income level affect savings behavior?

\- Which bank branches perform best?

\- What is the financial health score of each customer?
